This is a real-time cybersecurity monitoring dashboard powered by Machine Learning.

- **Instant Predictions**: Detects normal vs anomalous network traffic.
- **Explainable AI**: Visualizes feature importance for every prediction.
- **Interactive UI**: Adjust connection parameters and see live results.
